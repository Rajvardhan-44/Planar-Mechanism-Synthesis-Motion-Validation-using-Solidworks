# Mechanism Synthesis for Prescribed Motion of a Partial Circular Object

## Problem Statement

A partial circular object of radius **5 cm** must move through three prescribed orientations while satisfying the following conditions:

- Motion sequence:
  - **A → B**
  - **B → C**
  - **C → A** (2× faster, through B)

- The object's **center does not translate horizontally**.
- The center only moves **vertically**.
- The mechanism must use **only revolute or prismatic joints**.
- Object is symmetrical about horizontal when it is at position B.
- The program should remain **generic** for different geometric dimensions (Location of B and C).
